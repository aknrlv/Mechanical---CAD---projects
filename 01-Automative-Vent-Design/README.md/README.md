# Automotive Vent Design

This project represents a circular automotive air vent designed as a practice
CAD model for automotive interior components.

## Objectives
- Practice parametric CAD modeling
- Maintain symmetry and clean geometry
- Design a manufacturable plastic part

## Features
- Circular vent geometry
- Repetitive patterned slots
- Central mounting hole
- Fully parametric dimensions

## Software Used
- Onshape

## Notes
This model was created for learning and portfolio purposes.
